## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/hackerjamil343423/landing-page-service.git
cd landing-page-service
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) to view the project.

## Build for Production

```bash
npm run build
npm start
```

## Deploy on Netlify

1. Connect your GitHub repository to Netlify
2. Set build command: `npm run build`
3. Set publish directory: `out`
4. Deploy!

The project is optimized for static export and works perfectly with Netlify's deployment pipeline.
